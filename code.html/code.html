    Práctica 1: Programación Lógica Pura



    Introducción

    Este módulo implementa operaciones básicas para trabajar con bytes en diferentes representaciones. Los bytes pueden representarse en formato binario (como 8 bits) o en formato hexadecimal (como 2 dígitos hexadecimales).

    Representación de datos

    El módulo utiliza dos representaciones principales para los bytes:

    • Bytes binarios: Representados como listas de 8 bits. Cada bit es una estructura b(0) o b(1).
    • Bytes hexadecimales: Representados como listas de 2 dígitos hexadecimales. Cada dígito es una estructura h(X) donde X es un valor entre 0 y f.

    Funcionalidades

    El módulo implementa las siguientes funcionalidades:

    1. Verificación de listas de bytes
    2. Conversión entre representaciones binarias y hexadecimales
    3. Obtención de un bit específico de un byte
    4. Desplazamiento circular a la izquierda de listas de bytes
    5. Desplazamiento circular a la derecha de listas de bytes
    6. Operación XOR entre bytes

    Ejemplos de uso

    A continuación, se presenta un ejemplo de cada funcionalidad principal del módulo.

    Verificación de lista de bytes: 

    ?- byte_list([[h(a),h(5)], [b(1),b(0),b(1),b(0),b(1),b(0),b(1),b(0)]]).
false.

    Conversión entre representaciones: 

    ?- byte_convert([h(5),h(a)], BinByte).
BinByte = [b(0),b(1),b(0),b(1),b(1),b(0),b(1),b(0)]

    Conversión de listas de bytes: 

    ?- byte_list_convert([[h(a),h(b)],[h(c),h(d)]], BinList).
BinList = [[b(1),b(0),b(1),b(0),b(1),b(0),b(1),b(1)],
           [b(1),b(1),b(0),b(0),b(1),b(1),b(0),b(1)]]

    Obtener bits específicos: 

    ?- get_nth_bit_from_byte(s(s(s(0))), [h(5),h(a)], Bit).
Bit = b(0)

    Desplazamiento circular a la izquierda: 

    ?- byte_list_clsh([[h(5),h(a)], [h(2),h(3)]], CLShL).
CLShL = [[h(b),h(4)], [h(4),h(6)]]

    Desplazamiento circular a la derecha: 

    ?- byte_list_crsh([[h(b),h(4)], [h(4),h(6)]], CRShL).
CRShL = [[h(5),h(a)], [h(2),h(3)]]

    Operación XOR entre bytes: 

    ?- byte_xor([h(a),h(5)], [h(5),h(a)], Result).
Result = [h(f),h(f)]

                    PREDICATEbyte_list_clsh/2
                    Este predicado es cierto cuando CLShL es el resultado de un desplazamiento circular a la izquierda de la lista de bytes L.

                    El predicado utiliza dos cláusulas:

                    1. Si L es una lista de bytes binarios, convertir la lista a bits, realizar el desplazamiento circular a la izquierda y volver a convertir a bytes.
                    2. Si L es una lista de bytes hexadecimales, convertir la lista a bytes binarios, realizar el desplazamiento circular a la izquierda y volver a convertir a bytes hexadecimales.
                    La implementación utiliza el predicado bytes_to_bits/2 para convertir la lista de bytes a bits, el predicado rotate_left/2 para realizar el desplazamiento circular a la izquierda y el predicado bits_to_bytes/2 para volver a convertir los bits a bytes. El predicado byte_list_convert/2 se utiliza para convertir los bytes hexadecimales a bytes binarios.

                    Usage:byte_list_clsh(L,CLShL)

                    CLShL es el resultado de aplicar un desplazamiento circular a la izquierda a la lista de bytes L. Su implementación es: 

                    byte_list_clsh(L,CLShL) :-
    bytes_to_bits(L,AllBits),
    rotate_left(AllBits,RotatedBits),
    bits_to_bytes(RotatedBits,CLShL).
byte_list_clsh(L,CLShL) :-
